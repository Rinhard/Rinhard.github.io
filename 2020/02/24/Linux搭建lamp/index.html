一.检查系统

检查系统是非常重要的一步，不然有可能因为重复安装带来麻烦。

请以以下方法进行判断识别，如果未被识别则没有安装如果系统安装过，或者安装失败，请清理一下系统后，再继续安装。

1.确认Linux版本

# cat /etc/redhat-release
CentOS Linux release 7.7.1908 (Core)

2.检查是否安装Apache

可以用

apachectl -v
# apachectl -v
Server version: Apache/2.4.6 (CentOS)
Server built:   Aug  8 2019 11:41:18

也可以用

httpd -v
# httpd -v
Server version: Apache/2.4.6 (CentOS)
Server built:   Aug  8 2019 11:41:18

也可以用

(不推荐)

# rpm -qa | grep httpd
httpd-tools-2.4.6-90.el7.centos.x86_64
httpd-2.4.6-90.el7.centos.x86_64
httpd-manual-2.4.6-90.el7.centos.noarch

3、检查是否安装过Mysql

大多数教程检查mysql使用

service mysqld start

但在CentOS7之后，这个指令无法顺利运行

# service mysqld start
Redirecting to /bin/systemctl start mysqld.service

改用

systemctl restart mysqld.service #重启mysql服务
systemctl start mysqld.service #启动mysql服务
systemctl stop mysqld.service #停止mysql服务
systemctl status mysqld.service #查看mysql服务状态

通过

systemctl status mysqld.service

可以判断，但比较麻烦

而通过查询路径的方法进行判断就比较简单

# whereis mysql
mysql: /usr/bin/mysql /usr/lib64/mysql /usr/include/mysql /usr/share/mysql /usr/share/man/man1/mysql.1.gz   #查看mysql安装路径
# which mysql
/usr/bin/mysql #通过 PATH环境变量查找可执行文件路径，用于查找指向这个命令所在的文件夹。

3、检查是否安装过PHP

php -v

简单易用

二、安装Apache、PHP、Mysql

我们通过yum进行安装。

1.安装apache

可通过

yum -y install httpd

直到返回

......  
Complete!

表示安装成功。

2.安装php

yum -y install php

直到返回

......
Complete!

3.安装php-fpm

PHP-FPM(FastCGI Process Manager：FastCGI进程管理器)是一个PHPFastCGI管理器,提供了更好的PHP进程管理方式，可以有效控制内存和进程、可以平滑重载PHP配置。

yum -y install php-fpm

直到返回

......
Complete!

4.安装mysql

yum -y install mysql

直到返回

......
Complete!

另外7.2版本的Centos已经把mysql更名为mariadb了。

5.安装mysql-sever

yum -y install mysql-server
    Loaded plugins: fastestmirror, langpacks
       Loading mirror speeds from cached hostfile

    * base: mirror.lzu.edu.cn
    * extras: mirrors.nwsuaf.edu.cn
    * updates: mirrors.tuna.tsinghua.edu.cn
      No package mysql-server available.
      Error: Nothing to do

​

是的会返回错误！！！

因为CentOS 7+ 把mysql从默认程序列表中移除了，用mariadb代替。。。

两种解决方案
1.安装mariadb
 yum install -y mariadb
2.从官网下载mysql- server
# wget http://dev.mysql.com/get/mysql-community-release-el7-5.noarch.rpm
# rpm -ivh mysql-community-release-el7-5.noarch.rpm
# yum install mysql-community-server

然后需要确定，输入y,共两次，直到返回

......
Complete!

6.安装php-mysql

yum -y install php-mysql

直到返回

......
Complete!

三.安装基本常用扩展包

1.安装apache扩展包

yum -y install httpd-manual mod_ssl mod_perl mod_auth_mysql

直到返回

......
Complete!

2.安装PHP扩展包

yum -y install php-gd php-xml php-mbstring php-ldap php-pear php-xmlrpc php-devel

直到返回

......
Complete!

3.安装mysql扩展包

yum -y install mysql-connector-odbc mysql-devel libdbi-dbd-mysql

直到返回

......
Complete!

四.配置apache和mysql开机启动

apache指令

systemctl start httpd.service #启动apache
systemctl stop httpd.service #停止apache
systemctl restart httpd.service #重启apache
systemctl enable httpd.service #设置apache开机启动

若安装的是mariadb

# systemctl start mariadb.service #启动MariaDB
# systemctl stop mariadb.service #停止MariaDB
# systemctl restart mariadb.service #重启MariaDB
# systemctl enable mariadb.service  #设置开机启动

然后重启对应服务

五.配置mysql

初次安装mysql没有密码，首先设置密码，mysql的默认账户为root

设置mysql数据root账户的密码：

mysql_secure_installation

一直回车

出现如下时输入密码，linux中密码不显示，出入后回车再输入一次：

New password:

接下来还会有四个确认：

Remove anonymous users? [Y/n]
Disallow root login remotely? [Y/n]
Remove test database and access to it? [Y/n]
Reload privilege tables now? [Y/n]

直接回车即可。

六.测试环境

1.输入服务器IP,说明Apach测试成功

2.测试php

进入/var/www/html,写一个简单的php测试文件

#cd /var/www/html
#touch phpinfo-test.php 
#vim phpinfo-test.php

用vim打开后，按i 进入编辑模式，写入以下代码

<?php
phpinfo()
?>

然后按esc退出编辑模式，回到控制模式，输入

输入服务器IP/phpinfo-test.php，出现下图则成功。
